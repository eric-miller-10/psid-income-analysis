# psid-income-analysis

## Overview
This project analyzes household income using data from the Panel Study of Income Dynamics (PSID). The goal was to identify key factors that influence income and examine how these relationships change over time.

## Data Preparation
- Selected relevant variables such as employment, education, and household characteristics
- Cleaned and structured data for analysis
- Applied log transformation to income to improve model performance

## Methods
- Ordinary Least Squares (OLS) Regression
- Correlation Analysis
- Comparative analysis across different years

## Key Findings
- Employment-related variables were strong predictors of income
- Log transformation improved model stability and interpretability
- Some predictors changed in importance over time, suggesting evolving economic patterns

## Tools & Libraries
- Python / R
- pandas / tidyverse
- statsmodels / lm()
- matplotlib / ggplot2

## Key Takeaway
This project highlights how statistical modeling can be used to understand real-world economic patterns and how relationships between variables evolve over time.
